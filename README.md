# Kali Linux Lab

## Objective

This repository documents my Kali Linux installation, configuration, and the first steps of building my personal cybersecurity laboratory.

## Environment

### Hardware

- **Laptop:** Lenovo ThinkPad T490
- **Processor:** Intel® Core™ i5-8365U @ 1.60 GHz (4 Cores / 8 Threads)
- **Memory:** 16 GB RAM
- **Storage:** 477 GB NVMe SSD
- **Architecture:** x86_64 (64-bit)

### Operating System

- **Distribution:** Kali GNU/Linux Rolling
- **Installation Type:** Native installation (single operating system)
- **Kernel:** Linux 7.0.12+kali-amd64
- **Firmware Version:** N2IETA5W (1.83)

### Disk Layout

| Partition | Purpose |
|-----------|---------|
| `/boot/efi` | EFI System Partition |
| `/` | Root filesystem |
| `[SWAP]` | Swap partition |

### Purpose

This environment is used as my primary operating system for learning Linux, system administration, networking, ethical hacking, penetration testing, and cybersecurity.
